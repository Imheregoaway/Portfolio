# Portfolio — Anirudh Kanwar

Static one-page portfolio generated from `Anirudh_Resume.pdf`, styled with a layout vibe inspired by `dontboardme.com`.

## Run locally

Option A (quickest): open `index.html` in your browser.

Option B (recommended): run a local static server.

```powershell
cd "C:\Work\Project\Portfolio"
python -m http.server 5173
```

Then open `http://localhost:5173`.
